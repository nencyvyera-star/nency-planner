# 🎨 Web Design Studio

Ninja Nency's space for building websites fast & beautiful. 🥷

## 📁 Folder Structure
- `projects/` — each website gets its own folder here
- `templates/` — reusable starter files (HTML/CSS boilerplates)
- `assets/` — shared images, icons, fonts

## 🧰 Installed Tools
| Extension | What it does |
|---|---|
| Tailwind CSS IntelliSense | Class autocomplete for fast styling |
| Color Highlight | Shows colors inline in code |
| Image preview | Thumbnails in the gutter |
| Auto Rename Tag | Edits paired HTML tags together |
| HTML CSS Support | CSS autocomplete in HTML |
| Live Preview | Preview pages inside VS Code |
| Figma for VS Code | Pull designs into the editor |
| Live Server ✅ | Live-reload local server |
| Prettier ✅ | Auto-format code |

## 🚀 Quick Start
1. Copy `templates/starter.html` into a new folder under `projects/`
2. Right-click the HTML file → **Open with Live Server** (or **Show Preview**)
3. Edit and watch it update live ⚡

## 💡 Tips
- Use Tailwind via CDN (already in the starter) — no build step needed to start
- Format anytime with `Shift + Option + F`
